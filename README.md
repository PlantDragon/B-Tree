	O projeto trata-se de uma busca em banco de dados utilizando uma estrutura de dados do tipo B Tree (Árvore B).
	
	Ele consta de um simples menu com as opções de:
	1) Busca pelo primeiro nome do filme (ou das primeiras letras);
	2) Busca pelo ano de lançamento do filme;
	3) Sai do programa.

	Porém, a função "insert_tree" não funcionou corretamente, impossibilitando os testes das demais funções do programa. Apesar disso, os métodos foram implementados.
